# 二、2026年的不确定性

*   **不爆雷的定义？**
    
    *   搞定外部？
        
        *   搞定市场监督局
            
        *   搞定媒体
            
        *   搞定网络舆情（正面评价率）
            
        *   搞定大黑用户
            
    *   搞定内部？
        
        *   如果要规范，尺度放在哪里？
            
        *   如果要测尺度，拉扯要多久？
            
*   **探索新卖点的时间**
    
    *   A+平台卖点撤销+质检合规，首次测试**转化率约0.8%**，二次测试**转化率约0.3%**
        
*   **各平台针封控趋严**[《近期H5风控情况梳理》](https://alidocs.dingtalk.com/i/nodes/QPGYqjpJYrjZ32ENUzPZl6jm8akx1Z5N?cid=164590581%3A783037887&utm_source=im&utm_scene=person_space&iframeQuery=utm_medium%3Dim_card%26utm_source%3Dim&utm_medium=im_card&corpId=ding057c2ffd583c3504ffe93478753d9884)
    
    *   内容敏感词触发投诉
        
        *   出现“新人、时薪、底薪、变现”等高危词汇
            
        *   缺乏“收益以实际情况为准”等免责说明
            
    *   域名与链接管理问题
        
        *   直播短链直接暴露于社交环境，易被学员直接投诉
            
        *   域名集中使用，一封多业务受影响
            
*   **张震IP的不确定性**
    
    *   7月到期，是否继续续约，以及继续配合（目前快手找到张震，希望签约独家）
        
*   **AI线增长情况的不确定**
    
    *   AI写作增长偏难，长篇小说是否可在今年跑通？以及规模量级多大？
        
    *   AI短视频目前叠加了很多资源，上限多高？

# 四、迎接现实的战役

## 战役一：【合规闪电战】——以最快速度建立“可对外公示”的防火墙，赢得合法冲刺期

**核心：用1-2个月时间，快速搞定外部监管与内部规则，把“不爆雷”的定义和尺度明确下来，让后续冲刺尺度最大化**

| 目标 | 关键结果与策略 | 责任人 | 时间 |
| --- | --- | --- | --- |
| **1. 2月底前，与监管部门达成“合规逃生”共识。** | **KR1：** **与各地市场监督管理局敲定 “投诉案例数量/投诉处理SOP”**<br>**参考：杭州谋事，量子之歌** | **@康杰 + 吴菁** | 2月 |
|  | **KR2：** 准备并启动 **“媒体关系专项预算”**，与至少 **3家国家级/核心商业媒体** 建立正式沟通渠道，确保在出现负面舆情时，有权威通道进行澄清或降温。 | **@马长久+公关负责人** | 3月 |
| **2. 3月底前，完成全渠道营销素材的“换血”。** | **KR1：** 恢复 **“高转化安全素材模板”**，确保4月起投放，账户能够快速重启。 | **@杨蓉蓉** | 3月 |
|  | **KR2：** **完成A+平台卖点撤销后的有效转化测试。**0元转化率不低于 **2.5%** | **@李萍** | 3月 |

---

## 战役二：【现金流掠夺战】——实施超高强度业务冲刺

**核心：将已验证的“现金牛”业务的产能提到历史极限，填补成本窟窿和利润目标**

| 目标 | 关键结果与策略 | 责任人 | 时间 |
| --- | --- | --- | --- |
| **1. 4月起，将主播学院（第一曲线）的月净利润稳定在1500万以上。** | **KR1：** **“大黑用户”拦截系统升级至2.0。** 不仅拦截经济/身体高风险用户，更要利用数据模型，**识别并拦截“潜在高投诉倾向”用户**，将这部分用户的投诉成本前置规避。 | **@康杰 + 姚涣** | 持续 |
|  | **KR2：** **实施“超级人效”计划。** 在跑赢大盘的情况下，将AI助教覆盖率提升至**50%**，将精英助教的人均月产出提升 **50%**。 | **@李萍（销售） + 姚涣** | Q2起 |
|  | **KR3：** **复购“精准爆破”。恢复复购节奏**，**在最可能付费的节点（如课程中期、有积极作品产出时）**，由最优秀班主任进行“一对一”升单推荐，目标将**复购转化率提升150%+** | **@谢媛** | Q2起 |
| **2. 确保AI线（写作/短视频）的利润和增长** | **KR2：** **集中资源测试“长篇小说”或“AI短剧，AI漫剧”等1-2个新卖点** | **@马长久 + AI线负责人** | H1 |

---

## 战役三：【IP与流量堡垒战】——最大化锁定并利用现有稀缺资源，建立短期护城河

**核心：在张震IP可能流失、各平台封控趋严的不利环境下，最大化现有IP价值，并开辟新的、安全的流量阵地**

| 目标 | 关键结果与策略 | 责任人 | 时间 |
| --- | --- | --- | --- |
| **1. 全力保障张震IP在7月前贡献最大化价值，并准备备选方案。** | **KR1：** **启动“张震IP价值榨取”计划。** 围绕张震，设计 **高强度、短周期的直播转化专场**（如“毕业作品点评会”、“大师限时班”），目标在合约期内，由其直接贡献 **不低于3000万** 的额外GMV。 | **@涂丹丹（直播） + 市场** | Q2 |
|  | **KR2：** **同步启动“继任者IP孵化和签约”**。 在Q2前，至少签约或内部测试 **1-2个** 具有潜力的配音领域新IP（如喜播内部优秀导师或学员），以应对7月后的不确定性。 | **@大卫（IP孵化）** | H1 |
| **2. 开辟高安全性的新流量渠道，对冲平台风险。** | **KR1：** **加速“私域+直播”闭环建设。** 将更多资源投入 **视频号、快手** 等私域属性更强、监管环境相对宽松的平台直播，降低对单一公域流量的依赖。 | **@直播运营团队** | 全年 |
|  | **KR2：** **建立“域名与链接风控池”。** 准备至少 **3套** 可快速切换的域名和短链服务，确保单一渠道被封时，能实现24小时内无缝切换，业务不中断。 | **@技术运维 + 风控** | Q1 |

---

## 战役四：【新赛道侦察战】——敏捷测试，为大健康/兴趣赛道准备“火种”

**核心：不以2026年贡献利润为目标，只用最小代价+快速测试，验证可行性，为公司可能的“断臂转型”保留一个未来选项**

| 目标 | 关键结果与策略 | 责任人 | 时间 |
| --- | --- | --- | --- |
| **1. 用不超过XX万总预算，验证“大健康/兴趣”赛道的用户基础与付费模型。** | **KR1：** **禁止站外投放。** 100%利用喜播现有用户池（如已毕业、满意度高的有声学员）进行 **存量转化测试**，招募不超过 **1000名** 内测用户。 | **@新业务实验室** | H1 |
|  | **KR2：** 验证核心命题：在 **无“副业赚钱”承诺** 的前提下，**纯兴趣或健康类课程** 的完课率能否 **≥70%**、用户NPS能否 **≥40**、是否有 **≥10%** 的用户愿意为进阶服务或周边产品付费。 | **@新业务实验室** | Q3前 |
|  | **KR3：** **输出一份决定性报告。** 基于测试数据，在Q3末提交 **《大健康/兴趣赛道可行性论证与建议报告》**，明确给出 **“全力投入”、“维持探索”或“放弃”** 的结论，为董事会年底的战略决策（是否关闭教育业务）提供依据。 | **@新业务实验室 + 战略部** | Q3 |
