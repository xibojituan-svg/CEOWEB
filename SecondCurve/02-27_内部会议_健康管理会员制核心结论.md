# 02-27 内部会议核心结论：健康管理会员制探索

## 1. 第一曲线现状与战略调整 (基本盘)
- **现状剖析**：近期受短视频平台政策收紧与合规性打击影响，目前流量与单日收入面临挑战；同时发现，过去由于唯规模论而过度承诺，带来了合规隐患（如“退费高”、“大黑用户风险大”）。
- **调整方向**：
  - **红线底线**：严控“大黑用户”，规范前端销售话术，杜绝不合理的造梦与承诺。
  - **合理利润目标**：不再盲目追求不健康的营收爆发，将年度健康净利润目标设定在合理区间（约 5000万），在风险可控的基础上寻求稳健经营，避免像部分业内友商那样发生系统性崩盘。
  - **定位转变**：将第一曲线业务界定为提供稳定现金流的“基础牛”，未来不再将其作为谋求上市的主要抓手。

## 2. 第二曲线（健康管理会员制）痛点与机遇
- **切入点（痛点）**：
  - 高净值人群及慢病潜伏人群对个性化、针对性的健康管理有刚需（例如肥胖带来的血脂、高血压等代谢问题风险）。
  - 目前高端体检机构能提供详尽的体检报告与全方位诊断，但在长期的**“慢病管理”、“行为监督”及“日常执行”**上存在服务断层与空缺。
- **服务模式构想（干预闭环）**：
  - **底层认知**：医疗专业知识（如报告初筛与指标解读）必然将被AI大幅替代，人类角色的核心价值不再仅是认知差，而是**陪伴、情感价值、督促与强制力的约束**。
  - **AI与健康管家结合 (TLC逻辑)**：类似高定私教模式，结合健康饮食方案、心理疏导与运动规划（有氧+无氧），AI给出策略，健康管家负责通过回访打卡监督用户养成习惯并提供情绪价值。
- **硬件与数据监测应用**：
  - 不建议自建大型重资产硬件，倡导使用通用型“连续性动态监测设备”（如华为等智能手表、手环以测算卡路里及心率情况）。
  - 针对具体健康指标搭配轻便医用居家设备（如几十到一百元级的常规血糖测试仪、家用血压测量仪、体脂秤）。
  - 商业逻辑：硬件只是数据采集触角，**真正的商业模式壁垒在于基于持续追踪数据的解决方案输出（算法层）与随访服务监督（管家层）。**

## 3. 组织架构与团队激励机制探讨
- **痛点与困境**：
  - 第一曲线现有团队在多年的训练营模式下已经形成了极强的“肌肉记忆”和固化思维。如果直接要求老团队平移进行重度创新的健康业务探索，极易能力水土不服、左右互搏。
- **内部创业模型解法**：
  - **独立组织**：建立完全独立于一业务线之外的创新 MVP 团队，外部引入具备 AI Native 思维与健康管理专业知识的人才。
  - **资本与利益分配模型**：
    - 作为创新孵化项目，以虚拟股权或类期权机制进行长期激励设置。既提供长期的巨大利益绑定可能，又保障短期内团队不会因缺乏基本盘护城河而患得患失。未来跑通并产生高估值后，可以折算并入并寻求融资。
  - **未来资本故事重构**：如能跑出闭环，未来该集团面临的一、二级资本市场沟通时，将核心讲述第二曲线（大健康管线+AI体系）的估值与成长性故事。

## 4. 下一步 0-1 阶段验证破局点
- 继续稳住一曲线的利润盘，管理团队明确底线与短期战略收缩边界的共识。
- 着手梳理、落地符合利益共同体的激励方案，安抚骨干军心，构建透明清晰的长中短期收益愿景。
- 全力搭建第二曲线初创模型团队：设计跑通从“高端用户引流 -> 体检测评输入 -> AI方案自动生成 -> 人工管家督促执行” 的MVP内循环，并在短周期（3个月内）观察用户的长期付费粘性。
