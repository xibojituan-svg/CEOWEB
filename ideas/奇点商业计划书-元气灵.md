### 「元气灵」商业计划书（AI心理疗愈方向）

++元气灵创始人：大美、余明柯、赵鸿阳++

**一句话概括** 「元气灵」是一款帮助用户**消解负面情绪**的**AI陪伴产品**。你上传一段5-15s**视频**，AI精准识别你的情绪状态，并为你打开一扇治愈之门——可以选择“比惨大会”**小游戏与陌生人PK负面情绪**，在倾诉与共鸣中收获抚慰；也可以选择**AI聊天或声音疗愈推荐**，让每一次情绪波动都转化为自我关怀的契机。积分可解锁深度课程与更多互动时长，陪你一步步走出情绪低谷。

**目标用户及核心痛点**

目标用户：18岁以上有情绪疏解需求人群。

痛点：

_孤独的“情绪内耗者”_：Z世代与高压职场人，习惯隐藏情绪，缺乏安全、有趣的宣泄出口。他们想倾诉，但怕被评判；想被安慰，但不知如何开口。

_社交疲惫的年轻人_：传统社交需维持形象，而“比惨大会”提供了一种低心理负担、高共鸣感的互动方式——原来有人和我一样，甚至比我更糟，这反而带来奇妙的治愈感。

_疗愈内容选择困难者_：市面上的冥想、音频课程虽多，但与当下情绪脱节，用户不知该听什么。元气灵基于实时情绪精准推荐，让疗愈“正好你需要”。

匿名倾诉App（如树洞、一罐）：仅提供倾诉空间，缺乏反馈与互动，倾诉后依然空虚。

社交媒体发泄：可能遭遇负面评论，或担心隐私暴露。

心理咨询：成本高、门槛高，对轻度情绪问题“杀鸡用牛刀”。

游戏化心理产品（如Moodpath、造梦次元）：多为自我追踪，缺乏社交互动与即时情绪释放的乐趣。

核心局限：没有将“情绪表达”与“游戏化社交”结合，未能让负面情绪在轻松互动中被转化。

**不解决问题的深远影响**

情绪长期压抑导致**焦虑、抑郁倾向**加重，影响工作效率与人际关系。

**孤独感**加剧，形成“越孤独越不愿表达”的恶性循环。

错失通过轻度社交获得情感共鸣的机会，心理健康**“小问题”拖成“大麻烦”**。

**AI的核心优势：让情绪被看见，让游戏有温度**

**5.1 多模态情绪感知**

情绪检测Agent

小游戏Agent

AI聊天Agent

通过面部微表情分析（基于NIH 2025研究成果）、语音语调识别、图像情绪标签，AI以超80%准确率判定用户当下的愉悦度、唤醒度及具体情绪类别（如“疲惫式低落” vs “愤怒式烦躁”），为后续互动提供精准输入。

**5.2 “比惨大会”游戏机制**

匹配PK：检测情绪后，用户点击“小游戏”进入“比惨大会”，系统匹配一位当前情绪状态相似或相反的用户（或AI虚拟对手）。

轮流倾诉：双方各用30秒语音或文字讲述自己此刻的负面情绪（如“今天被老板骂了” vs “失恋第三天”）。

AI裁判打分：系统基于情绪强度、表达深度、关键词等维度，给出“惨度指数”（如7.2分 vs 8.5分）。

胜负机制：

赢家（心情值增加）：发现对方比自己更惨，产生“原来我不是最糟的”心理慰藉，心情值+1。

输家（获得抚慰积分）：虽败犹荣，获得抚慰积分（可累积解锁课程/AI聊愈时长），并立即收到AI基于其倾诉内容定制生成的安慰文字（如“被否定一定很难过，但你的价值不由他人定义”）。

情绪价值：将“比惨”这一民间自嘲行为游戏化，让用户在欢笑与共鸣中释放情绪，同时获得切实的抚慰激励。

**5.3 AI聊天与声音疗愈**

AI聊天：若不想社交，可选择与AI倾诉。AI基于情绪识别，以共情式对话引导用户梳理感受，并适时提供认知行为疗愈建议。

声音推荐：根据情绪状态，推荐精准匹配的疗愈音频（如焦虑时听432Hz颂钵、疲惫时听森林白噪音），支持实时生成个性化声景。

**5.4 积分激励闭环**

抚慰积分可用于：

解锁线上声音聊愈课程（如“7天走出低自尊”“失眠自救指南”）

兑换AI聊愈互动时长（深度对话模式）

购买实体疗愈礼盒（未来扩展）

积分机制鼓励用户持续使用“比惨大会”，形成“倾诉-获积分-学习-成长”的正向循环。

前期用户量不高的情况下，完全可以通过云平台的免费额度来完成，即使超过了免费额度，每次调用的成本也就在几分钱左右。

**5.5 技术可行性的坦诚承诺**

> “关于面部情绪识别技术，我们进行了系统的学术与工程调研。**结论是：这条路是通的，且有扎实的科学依据**。2025年NIH的最新研究已证明，AI通过面部动作单元分析可有效推断情绪状态。**但是，我们也清醒地认识到技术的边界**：‘皮肤状态、发质’等静态特征与即时情绪的关联尚无科学共识。因此，**元气灵MVP的核心突破点是‘情绪识别+自适应声音疗愈’**，这是我们有信心在3个月内完成技术验证的路径；‘抗衰影像分析’和‘系统课程体系’将作为**第二阶段功能**，以更严谨的方式逐步上线。**坦诚技术的能与不能，恰恰是项目长期信任的基石。**”

**MVP验证方案**

产品形态：微信小程序（优先），支持语音/文字输入，基础面部情绪识别（使用开源模型简化版）。

核心验证场景：“情绪检测→比惨大会→积分获取”3分钟闭环

情绪检测：用户上传一段15秒视频/语音/图片，AI生成简易情绪标签（如“焦虑”、“疲惫”）。

游戏匹配：用户点击“比惨大会”，系统随机匹配在线用户或AI模拟对手。

倾诉与打分：双方各输入30秒语音/文字，AI展示“惨度指数”及胜负结果。

激励发放：

赢家心情值可视化提升（UI动画）

输家获得抚慰积分+定制安慰文字（可保存分享）

积分兑换预览：展示“可用积分兑换课程”的入口，但不强制转化。

验证关键指标：

✅ 游戏参与率：完成情绪检测后，点击“小游戏”的比例（目标≥60%）

✅ 游戏完成率：进入匹配后，完整完成一轮倾诉的比例（目标≥80%）

✅ 用户回访率：次日/7日后再次使用“比惨大会”的比例

✅ 情感共鸣度：用户对“对方让我感觉好多了”或“安慰文字很温暖”的评分（≥4.0/5）

✅ 积分激励有效性：获得积分后，查看课程详情或尝试兑换的比例

✅ 分享率：生成惨度海报并主动分享的比例（目标≥20%）

**游戏化呈现方式**：

### 一、核心设计原则

| 原则 | 说明 |
| --- | --- |
| **去评判化** | 不是评判“你的烦恼值不值得”，而是“你此刻的感受有多强烈” |
| **多维融合** | 结合AI情绪检测数据+用户语言表达内容+表达方式 |
| **正向激励** | 分数越高（越惨）不代表“你更惨”，而是“你更需要被抚慰” |
| **游戏化包装** | 用趣味指数包装，避免冰冷打分感 |

### 二、烦恼指数评分维度

AI将从以下四个维度综合计算“惨度指数”（满分10分）：

#### 维度1：情绪强度（AI视觉+语音分析）—— 权重40%

**面部表情强度**：基于NIH研究的AU动作单元分析，检测悲伤、愤怒、疲惫等情绪的肌肉运动幅度

**语音特征**：语速（低落时通常变慢）、音调起伏、颤抖度、停顿频率

**生理代用指标**：如用户授权可穿戴设备，可参考心率变异性（HRV）等

**示例**：

轻度低落 → 3-4分

明显悲伤表情+语音颤抖 → 7-8分

#### 维度2：语言内容分析（NLP）—— 权重35%

**关键词强度词典**：建立“烦恼关键词库”，分级赋分

1-3分：有点烦、好累、不想动

4-6分：难过、委屈、失望、被忽视

7-9分：崩溃、绝望、撑不下去了、失去意义

**事件类型系数**：

日常琐事（迟到、外卖洒了）→ 系数0.8

人际关系摩擦 → 系数1.0

重大失落（分手、亲人健康、职业危机）→ 系数1.2-1.5

#### 维度3：表达丰富度—— 权重15%

**字数/时长**：通常越深的烦恼，用户会说得越多（但也有人欲言又止，需综合判断）

**细节程度**：是否有具体场景、感受描述

**重复强调词**：“真的真的很难过”等

#### 维度4：与历史基线对比—— 权重10%

对于老用户，对比其**历史平均情绪值**：如果用户平时很开心，今天突然很低落，则指数会调高（“你今天比平时低落很多哦”）

对于新用户，使用常模参照

### 四、比惨大会中的动态对战规则

在“比比皆丧”对战中，AI裁判会基于上述维度计算双方“惨度指数”，并执行以下机制：

#### 胜负判定逻辑

| 情况 | 赢家（心情值+1） | 输家（获抚慰积分+定制安慰） |
| --- | --- | --- |
| 你8.5分 vs 对方6.2分 | ❌ 对方 | ✅ 你（更惨但被安慰） |
| 你4.3分 vs 对方7.8分 | ✅ 你 | ❌ 对方 |
| 你6.5分 vs 对方6.8分 | ❌ 对方（微弱优势） | ✅ 你（虽败犹荣） |

#### 特殊规则

**“同病相怜”加成**：如果双方烦恼类型相似（如都是职场压力），两人**额外各得5抚慰积分**，并弹出：“你们是同类！”

**“反向惨胜”彩蛋**：如果赢家发现自己赢了但对方确实很惨，可主动选择“分一半心情值给对方”，触发隐藏成就

**每日“惨王”称号**：当天惨度指数最高的用户，获得“今日惨王”徽章，积分翻倍

### 五、AI定制安慰文字的生成逻辑

输家获得的安慰文字，不是随便写的——它基于以下要素**动态生成**：

**烦恼类型识别**：从用户表达中提取关键词，判断属于“职场/情感/健康/自我怀疑”等类别

**强度匹配**：越惨的等级，安慰语越深沉有力

**人格化适配**：根据用户历史互动记录，选择ta偏好的安慰风格（温暖型/幽默型/哲理型）

**示例**：

输入：“今天方案被毙了，老板说我没脑子”😩

AI识别：职场否定+自我怀疑，强度7分（暴雨级）🌧

生成安慰：“被否定确实像被浇了一盆冷水。但你知道吗？爱迪生也被说过‘没脑子’。你的价值，不由任何人的一句话决定。”

### 六、用户体验保护机制

为防止滥用或负面体验，必须设置以下规则：

**最低表达门槛**：少于5个字或纯表情包，不予匹配（“说一句嘛，不然没法比”）

**敏感词拦截**：涉及自伤、他人伤害等真实危机，触发心理援助资源推荐，不进入比惨游戏

**每日上限**：每人每天最多玩10次“比比皆丧”，防止情绪透支

**冷却机制**：连续输3次后，强制推荐AI聊天或声音疗愈（“今天已经够惨了，休息一下吧”）

### 七、产品话术建议（UI展示）

| 场景 | 话术 |

| 评分展示 | “你的烦恼指数：**暴雨级难过 7.8分**” |

| 输家提示 | “你赢了...同情分？抚慰积分+15，抱抱你 🌧️” |
| 安慰卡片标题 | “给暴雨中的你” |
| 分享海报文案 | “今天在@元气灵 比惨，我惨到了8.5分，居然有点爽？” |

这套规则既保留了AI情绪识别的科学底色，又用游戏化包装让“比惨”变得轻松可接受。它让用户感受到：**我的情绪被认真量化了，但这种量化不是为了评判我，而是为了更好地安慰我。**
