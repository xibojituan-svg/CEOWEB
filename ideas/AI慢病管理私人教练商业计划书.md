# AI慢病管理私人教练商业计划书

# 「吃对了」—— AI慢病管理私人教练 MVP商业计划书

## 一、一句话概括项目

**为中国3亿+慢性病患者（糖尿病、高血压、痛风等）提供AI驱动的日常管理教练——用对话式AI替代"每季度见一次医生"的低频模式，让患者每天都有一个懂他病情、盯他执行的私人健康教练。**

产品名称：**「吃对了」**——慢病管理的关键战场就是每一餐，"吃对了"既是产品名，也是用户每餐打开它之后想得到的那句确认。

## 二、问题与痛点

### 核心问题

中国确诊慢性病患者超3亿（糖尿病1.4亿、高血压2.45亿、高尿酸/痛风1.77亿），慢病管理的本质是**365天的日常自律**——饮食控制、规律用药、运动达标、指标监测——但现实是绝大多数患者"管不住自己"。

### 用户现状

| 现有方案 | 问题 |
| --- | --- |
| 每季度去医院复查 | 医生只能看到检查日的数据快照，对过去90天的生活管理一无所知；门诊3分钟，来不及细聊 |
| 自己搜信息/看科普 | "糖尿病能吃西瓜吗"搜出100种相反答案；缺乏个性化，不知道"我这个情况"该怎么办 |
| 买血糖仪/血压计自测 | 数据记了一堆，没人帮忙解读，不知道趋势意味着什么 |
| 家人监督 | "你又吃多了""你怎么不运动"——变成家庭矛盾源头 |
| 私人营养师/健康管理师 | 有效但贵（500-2000元/月），且覆盖不到日常每一餐、每一刻 |

### 问题严重性

*   糖尿病患者血糖达标率仅**15.8%**，高血压控制率仅**16.8%**
    
*   不达标直接后果：并发症（糖尿病肾病、视网膜病变、心脑血管事件），医疗费用指数级上升
    
*   一个II型糖尿病患者的年均直接医疗费用约**9000元**，发生并发症后飙升至**3-5万元**
    
*   **患者不是不想管，是缺乏持续的、个性化的、低门槛的日常指导和监督**
    

## 三、AI原生解决方案（为什么AI是核心）

传统慢病管理失败的根本原因是**"高频需求 vs 低频服务"**——患者每天需要管理决策，但医生/营养师只能提供间歇性服务。AI天然解决这个矛盾：

| 能力 | 传统方案 | AI方案 |
| --- | --- | --- |
| 响应速度 | 挂号排队数天 | 即时回答，7×24小时 |
| 个性化程度 | 医生记不住每个患者细节 | AI记住所有病史、用药、饮食偏好、生活习惯，越用越懂你 |
| 覆盖密度 | 每季度1次门诊 | 每餐、每天、每次用药都可以互动 |
| 数据整合 | 纸质病历+零散检查单 | 自动整合血糖/血压趋势、饮食记录、运动数据，形成全景视图 |
| 行为干预 | "少吃甜的多运动"笼统建议 | "你今天午餐碳水已经65g了，晚餐建议选XX，控制在40g以内" |
| 情绪支持 | 无 | 理解患者"管不住嘴"的挫败感，不批判、给鼓励、帮调整 |

**AI不是替代医生，而是填补"两次门诊之间的90天真空"。**

核心AI能力：

1.  **饮食即时评估**：拍照/文字描述一餐，AI秒级反馈营养结构和对血糖/尿酸/血压的影响预估
    
2.  **指标趋势解读**：连接血糖仪/血压计数据，AI识别波动规律，给出原因分析和调整建议
    
3.  **个性化方案动态调整**：根据用户的实际执行情况，AI不断微调饮食/运动/作息建议（而非一成不变的"标准方案"）
    
4.  **用药管理与提醒**：结合用药方案和实际指标，AI判断用药依从性并智能提醒
    

## 四、目标用户

**首期聚焦：II型糖尿病患者**（1.4亿人，管理频率最高、饮食控制需求最强）

| 用户分层 | 画像 | 核心需求 | 获客渠道 |
| --- | --- | --- | --- |
| 核心用户 | 30-60岁，确诊糖尿病1-5年，正在用药/打针 | 每餐饮食把关+指标解读+用药提醒 | 病友群/药房/社区卫生中心 |
| 扩展用户 | 25-45岁，体检查出血糖偏高/高尿酸/轻度高血压 | 日常饮食管理+预防恶化 | 小红书/抖音健康内容 |

**后续扩展**：痛风（高尿酸管理）→ 高血压 → 高血脂，逐步覆盖全慢病谱。

## 五、产品形态（MVP）

**微信小程序「吃对了」**，核心功能3个：

### 1. AI饮食教练（核心钩子）

*   用户描述/拍照一餐 → AI即时反馈：营养成分估算、对你的指标影响评估、优化建议
    
*   示例："中午吃了一碗牛肉面+一瓶可乐" → "这餐碳水约95g（建议单餐＜60g），可乐含糖53g直接影响餐后血糖。建议：面量减半+加一份青菜，用无糖饮料替代"
    
*   支持语音输入（中老年友好）
    

### 2. 指标记录与AI解读

*   手动输入或蓝牙连接血糖仪/血压计，记录每日指标
    
*   AI自动生成趋势图+解读："你近7天空腹血糖均值从7.2降到6.8，午餐控碳水的策略有效，继续保持"
    
*   **可导出月度报告给主治医生**——让门诊3分钟变得高效
    

### 3. 每日AI简报

*   早上推送：今日饮食建议+用药提醒+运动目标
    
*   晚上推送：今日管理小结+明日调整建议
    
*   节假日/聚餐场景特别提醒（如"春节聚餐控糖指南"）
    

### 技术架构

*   **前端**：微信小程序（低门槛，中老年友好）
    
*   **AI引擎**：LLM（饮食理解+个性化建议生成）+ 营养数据库RAG（保障专业准确性）
    
*   **数据层**：用户画像（病种、用药、饮食偏好、指标历史）持续积累，驱动个性化

## 七、如何0新增HC服务1000名种子用户

**核心策略：利用存量渠道+病友社群的天然裂变能力，0人力成本触达目标用户。**

| 阶段 | 策略 | 预期获客 |
| --- | --- | --- |
| 第1波 | **内部员工+亲友** — 公司内部及员工家属中有慢病人群（高血压/糖尿病/痛风在30岁以上人群中极其普遍），发起"帮爸妈/帮自己管好慢病"内部活动 | 100人 |
| 第2波 | **病友社群渗透** — 在糖尿病/痛风/高血压贴吧、豆瓣小组、微信病友群中，以"免费AI营养师帮你看每餐能不能吃"为钩子吸引试用；产出10篇"AI帮我控糖30天实录"的真实UGC投放小红书/抖音 | 400人 |
| 第3波 | **医院/药房合作** — 与社区卫生中心、连锁药房合作，在慢病患者取药/复查时推荐"免费AI慢病管理工具"（放置二维码物料即可，无需人力驻场） | 300人 |
| 第4波 | **裂变机制** — "邀请病友一起管理，双方各得7天会员"；慢病患者天然有"病友圈"社交关系，裂变系数高 | 200人 |

**关键点：慢病人群有3个天然优势使其获客成本极低：**

1.  **痛点持续存在**——不像急性病好了就走，慢病患者永远需要管理
    
2.  **社群化程度高**——病友群、病友论坛、病友互助是刚需社交场景
    
3.  **信息饥渴**——"XX能不能吃"是每天都会问的问题，AI即时回答是强钩子
    

## 八、MVP验证方案（8周）

### 第1-2周：需求验证

*   **方式**：在3-5个糖尿病/痛风病友群发放问卷（目标300份）+ 深度访谈15位慢病患者
    
*   **核心验证问题**：
    
    *   你每天在"能不能吃XX"上花多少时间？对答案满意吗？
        
    *   你的指标达标率如何？你觉得主要卡在哪？
        
    *   如果有个AI每天帮你看每餐吃得对不对、指标趋势好不好，你会用吗？
        
*   **通过标准**：
    
    *   ＞70%每周至少3次纠结"能不能吃"
        
    *   ＞60%自认指标管理不达标
        
    *   ＞50%对"AI饮食教练"概念感兴趣
        

### 第3-5周：构建MVP并冷启动

*   **开发重点**：
    
    *   搭建慢病饮食知识库（糖尿病/痛风/高血压 TOP 200食物的营养数据+影响评估）
        
    *   AI饮食评估引擎（LLM + 营养数据库 RAG）
        
    *   指标记录+趋势可视化
        
    *   每日AI简报推送
        
    *   电商导购模块（对接京东/淘宝联盟健康品类）
        
*   **冷启动**：
    
    *   招募100名种子用户（糖尿病50人+痛风30人+高血压20人）
        
    *   建立3个病种的种子用户微信群，产品经理每天跟进反馈
        
*   **通过标准**：
    
    *   MVP上线并稳定运行
        
    *   AI饮食评估准确率＞85%（由营养师抽检）
        
    *   种子用户100人入驻
        

### 第6-7周：验证留存与行为改变

*   **追踪核心指标**：
    
    *   日活跃率（目标＞35%，慢病管理是每日需求）
        
    *   单用户日均使用次数（目标＞2次，至少记录2餐）
        
    *   7日留存率（目标＞30%）
        
    *   用户主观感受：通过问卷验证"是否觉得饮食管理变容易了"（目标＞60%正向）
        
    *   **指标改善信号**：种子用户自报的血糖/尿酸/血压趋势是否有改善迹象
        
    *   **电商转化**：商品推荐点击率+成交率
        
*   **通过标准**：
    
    *   留存达标
        
    *   ＞50%用户反馈"AI帮助我做出了更好的饮食选择"
        
    *   出现用户自发分享案例
        

### 第8周：验证付费意愿

*   推出"吃对了Pro"会员（19.9元/月）：
    
    *   解锁无限AI对话（免费版每日5次）
        
    *   个性化周报/月报（可导出给医生看）
        
    *   用药提醒+医嘱管理
        
    *   全部健康商品AI导购
        
*   **通过标准**：付费转化率＞8%
