# 2026年喜播教育业务OKR（理想主义版）

![2026年BSC.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5boNZ8WNlxo/img/33be7615-5137-42f5-8e7e-e03f2f62d1df.png)

本OKR体系旨在将BSC指标转化为各部门可追踪、可考核的具体目标和关键成果，并强化跨部门协同。

**1. 市场与增长部 (承接BSC：客户、财务维度，负责人：@马长久 @杨蓉蓉 @涂涂)**  
市场部需从“流量采购者”转型为“健康增长与用户洞察的驱动者”。

*   **O1：【提升获客质量，保障财务健康线】**
    
    *   **KR1：** 将“百万元投诉率”作为核心健康指标，**同比下降10%**。行动：建立投放素材-销售承诺-课程交付的三方对齐审核机制，每月复盘投诉TOP10的素材与流量渠道。
        
    *   **KR2：** 在首购UE模型**净利率≥0%**的约束下，稳定获取有效流量（Leads），确保年度收入目标达成。行动：重点优化站内及抖音、小红书等已验证渠道的投放效率，将CAC控制在目标范围内。
        
*   **O2：【探索新增长曲线，为新品类孵化奠基】**
    
    *   **KR1：** 完成至少 2个新品类（如大健康/兴趣类）的MVP课程0-1市场验证，**3个月内单月GMV达到100万**测试门槛。
        
    *   **KR2：** 基于VOC（用户之声）和数据洞察，每季度输出 **《市场机会洞察报告》**，为新课程研发提供至少2个明确方向建议。
        

**2. 销售与转化部 (承接BSC：客户、内部运营维度，负责人：@马长久 @李萍)**  
销售部从“收割者”转型为“价值传递与信任建立者”，核心是长周期服务用户。

*   **O1：【严控销售风险，重塑用户信任】**
    
    *   **KR1：** 将销售团队的绩效**与 “所服务用户的百万元投诉率”** 强挂钩，**权重不低于30%**，引导销售过程回归真实价值传递。
        
    *   **KR2：** 100%推行新版合规销售SOP，并通过AI质检工具实现高危违规话术（如保证收益）的实时拦截，**违规率降至 5% 以下**。
        
*   **O2：【提升转化效率与长期价值】**
    
    *   **KR1：** 通过AI助教/班主任等工具，将标准化服务覆盖率提升至 **80%**，实现**前端人效提升 50%**。
        
    *   **KR2：** 将首购用户 **“开营后14天满意度/NPS”** 纳入核心考核，目标值 **≥X**，并与销售激励直接关联。
        

**3. 教研与产品部 (承接BSC：内部运营、学习与成长维度，负责人：@马长久 @苏秦)**  
教研产品部是健康交付和价值落地的核心，需加速AI融合与敏捷迭代。

*   **O1：【交付高质量的课程产品，提升完课与毕业标准】**
    
    *   **KR1：** 明确并提升 “学员合格毕业率”，确保**毕业学员技能达标率 ≥70%**，**并建立可验证的技能评估标准。**
        
    *   **KR2：** 基于2025年VOC恳谈会反馈，**Q1完成核心课程（如攀登计划L1/L2）的针对性重构**，重点解决“学不会”、“耗时过长”问题。
        
*   **O2：【加速产品创新与AI融合，孵化第二曲线】**
    
    *   **KR1：** 确保全年有 **2个新课程成功上线并达到卓越/优秀标准**（6个月时间单月GMV过300万，UE健康）。
        
    *   **KR2：** 将AI工具深度融入课程设计、教学服务与学员测评流程，实现敏捷研发，**课程研发效率提升100%**
        

**4. 服务与交付部 (承接BSC：客户、内部运营维度，负责人：@马长久 @谢媛 @佀曼曼)**  
服务交付部是“学员成功”的最终保障，必须建立从学习到变现的闭环。

*   **O1：【构建“毕业不离校”的长期价值生态】**
    
    *   **KR1：** 启动并跑通 “结业后3个月轻陪伴计划”，将结业学员**3个月内活跃留存率提升至40%**。
        
    *   **KR2：** **加速交付撮合平台迭代**，实现学员能力画像与商单的初步智能匹配，确保**毕业学员接单成功率≥5%**。
        
*   **O2：【以数字化和AI驱动服务效率革命】**
    
    *   **KR1：** 优化班主任激励模型，降低复购金额的直接挂钩比例，提升 **“学员满意度”、“商单撮合成交率”** 等指标权重。
        
    *   **KR2：** 基于服务数据，建立 **“学员退费风险预警模型”** 并于Q2上线，实现主动干预，**将因服务问题导致的退费率降低 20%**。
        

**5. 产研与数据部 (承接BSC：内部运营、学习与成长维度，负责人：@姚涣(Aaron姚涣))**  
产研部是支撑业务健康转型的技术底座，核心是提效和赋能。

*   **O1：【以AI工具全面提效，支撑业务健康化运营】**
    
    *   **KR1：**  将已验证的AI助教/班主任能力规模化推广，在2026年实现标准化服务环节的人力替代率40%，并将因此**节省全年净利润2%**的成本。
        
    *   **KR2：** 完成 **“学员能力画像”系统V1.0** 开发并上线，实现**商单智能匹配成功率提升至50%**，初步验证“数据-能力-变现”的闭环。
        
*   **O2：【搭建关键业务平台，保障战略落地】**
    
    *   **KR1：** 确保交付撮合平台及其 **“内部投诉与反馈通道”** 在Q3前全面上线并实现与客服、质检系统打通，**外部投诉率降低30%**。
        
    *   **KR2：****实现业务流量100%自动分配**，打通从leads-转化-服务的数据闭环。

#### 关键执行保障与协同机制

**季度评审与动态调整：** 上述OKR每季度需进行复盘，根据业务健康度（退费率、投诉率）和财务基线完成情况，动态调整资源优先级。

**跨部门协同项目制：** 针对“新课程上线”、“交付平台建设”等关键KR，必须成立由市场、教研、产研、服务等多部门组成的虚拟项目组，由一个主负责人领导，并对小组成员绩效全权负责，打破部门墙。

**激励机制改革：** 人力资源部需在Q1前完成薪酬结构改革方案，将新课程，重大改造项目与个人绩效强绑定，从根源上驱动行为转变。
