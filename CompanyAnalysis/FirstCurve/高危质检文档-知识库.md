# 高危质检文档-知识库

## 一、高危质检规则说明

高危质检规范按管理范畴划分为**员工管理类、业务规范类、监管风险类**三大类别，各类别规则的制定、调整、增减均由对应归口部门审定决策，具体如下：

*   员工管理类（3项）：制定、调整、增减，由**综合管理部**审定决策
    
*   业务规范类（11项）：制定、调整、增减，由**学员成功部**审定决策
    
*   监管风险类（16项）：制定、调整、增减，由**客户体验部**审定决策
    
    质检合规组负责全部质检规范条例的日常运营，如发布宣贯、日常执行与效果监控；若遇规则解读争议，按规则所属类别提请对应归口部门决策，质检合规组需及时响应各相关部门的工作需求。

    | **B类红线** | **协助伪造退费证据** | 禁止协助学员伪造材料、虚构理由等不正当方式申请退费，包括但不限于：<br>① 指导学员伪造病例、经济困难证明、学习障碍说明等虚假材料；<br>② 教唆学员伪装疾病、失业、家庭变故等理由博取同情，达到退费目的。 |
    | **B类红线** | **刷单/引导退费** | 禁止虚构订单或引导学员退费的操作，包括但不限于：<br>①自购或者让关联方下单购买课程；<br>②引导教唆已购买的学员退费，来自己处买单（含前后端、跨产品线）；<br>③诱导学员先买后退，在直播间主动提及“退费”、“学完不满意再退”等负面字眼。 |
    | **B类红线** | **压单/拖延退费** | 禁止学员提出退费申请后，为维护个人业绩而故意压单、拖延处理。具体违规行为包括：<br>①引导学员先延期再办理退费，过了某个时间再退；<br>②恶意拖延，如假装不知道如何退费， 谎称对公转账周期很长15-20天；<br>③故意失联：看到消息不回复或者过了好几天才回复（账号异常除外）；<br>④推诿处理：自己不处理， 让找高阶延期班主任或者客服；<br>⑤设置退费障碍：“退费需要扣手续费”“已开课不能退”。 |
    | **B类红线** | **喜马相关表达** | 禁止使用以下喜马拉雅相关的违规表述或素材，包括但不限于：<br>①展示、使用喜马任何官方 logo、图标、品牌标识及官方客服热线、邮箱；②以 “喜马官方”“喜马直属”“喜马总部” 名义进行招募、课程宣传；③自称 “喜马员工”“喜马官方讲师”，或表述 “喜播与喜马为同一主体”“课程由喜马直接运营” 等误导性内容。 |
